# pneumonia
🫁 Pneumonia Detection – Logistic Regression

📂 Loads patient data from pneumonia_dataset.csv
🔍 Explores and cleans the data (null checks, stats)
✂️ Splits into 80/20 train/test sets & scales features
🤖 Trains a Logistic Regression classifier
🎯 Evaluates accuracy + confusion matrix
🧪 Tests on 3 sample patients (Sick, Healthy, Unsure)
📊 Visualizes results with matplotlib charts

Stack: Python · pandas · scikit-learn · matplotlib
